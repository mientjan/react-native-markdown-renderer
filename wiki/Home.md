# React Native Markdown Renderer

A 100% CommonMark-compatible markdown renderer for React Native using native components (not WebView). All elements are rendered as native React Native components that can be overwritten when needed.

[![npm version](https://img.shields.io/npm/v/react-native-markdown-renderer.svg)](https://www.npmjs.com/package/react-native-markdown-renderer)

## Pages

- [Getting Started](Getting-Started) - Installation, requirements, and basic usage
- [Custom Styles](Custom-Styles) - Override default styles for any element
- [Custom Rules](Custom-Rules) - Replace render rules with your own components
- [Custom Renderer](Custom-Renderer) - Provide a fully custom AstRenderer
- [Plugins](Plugins) - Extend syntax with markdown-it plugins
- [API Reference](API-Reference) - Complete API documentation
- [Migration from v3](Migration-from-v3) - Upgrading from v3 to v4

## Example App

The repository includes an Expo example app with interactive demos of each feature. See the [`example/`](https://github.com/mientjan/react-native-markdown-renderer/tree/master/example) directory.
